Delightful UI Elements

A series of illustrations to inject some fun into the CloudFlare dashboard.

CloudFlare's dashboard is massive, feature-rich, and highly-interactive. The tools are powerful and their controls and settings can be complex. But that doesn't mean that it has to be boring. Adding illustrations to the UI added some color and fun.

Welcome tour

To ease cutomers into the dashboard redesign, I created a simple, multi-page modal that served as a tour to the new UI as well as a small introduction to our new style.

By reducing the UI elements to blocks of color a layer of complexity is stripped away and the concepts are simpler and easier to read and understand quickly.

Plan upgrades

The CloudFlare dashboard allows customers control their plan settings and plan levels. Plan levels can be changed at any time, although the CloudFlare enterprise plan requires users to request a quote.

To add a little bit of fanfare to the final confirmation modal, I created illustrations for plan level changes, as well as quote requests. Users know that their change has been processed, or their request sent. They're also reminded of billing and feature changes.
